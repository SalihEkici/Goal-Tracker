# Goal Tracker App

This is a mobile application built with React Native and Expo that allows users to set and achieve their personal and professional goals. The app provides an intuitive interface for adding and keeping track of their goals.

## Table of Contents

- [Getting Started](#getting-started)
- [Installation](#installation)
- [Future Enhancements](#future-enhancements)

## Getting Started

Follow these instructions to get a copy of the project running on your local machine for development and testing.

### Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/SalihEkici/Goal-Tracker
   cd goal-tracker-app
   ```

2. Run the application:

   ```
   npm start
   ```

3. View the app on your device via Expo go

## Future Enhancements

1. **Goal Time Tracking**: Track and view the time spent on each goal to gain insights into productivity and time management.

2. **Experience System**: Implement an experience (XP) and leveling system where users earn XP for completing goals, unlocking higher levels and potentially new features to increase motivation.

3. **Weekly Time Analytics**: Add graph-based visualizations to show the time spent working on goals throughout the week, helping users identify trends and improve time allocation.
